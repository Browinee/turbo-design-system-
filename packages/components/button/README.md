# Button


## Install

| type | command                                        |
| ---- | ---------------------------------------------- |
| npm  | `npm install @turbo-design-system/button` |
| yarn | `yarn add @turbo-design-system/button`    |
| pnpm | `pnpm add @turbo-design-system/button`    |


## Usage

```jsx
import Button from `@turbo-design-system/button`;

function App() {
	return (
		<Button />
	);
}
```
